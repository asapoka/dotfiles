# dotfiles

## Installation

Linux and macOS

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/asapoka/dotfiles/master/install.bash)"
```

Windows

```pwsh
cd dotfiles/Powershell
sudo install.ps1
```
