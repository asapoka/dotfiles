# dotfiles

## Installation

linux and macOS

```bash
cd dotfiles
bash install.bash
```

Windows

```pwsh
cd dotfiles/Powershell
sudo install.ps1
```

## Brewfile の書き出し方(usr ディレクトリ配下に .Brewfile を作成する)

```

brew bundle dump --force --describe --global

```
